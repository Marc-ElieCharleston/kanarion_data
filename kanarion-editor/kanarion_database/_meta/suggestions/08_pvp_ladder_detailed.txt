                    1. VUE D'ENSEMBLE DU SYSTEME LADDER
================================================================================

FORMATS PROPOSES
----------------
Apres analyse, les formats les plus pertinents pour Kanarion:

| Format | Type        | Priorite | Raison                              |
|--------|-------------|----------|-------------------------------------|
| 2v2    | Duo         | HAUTE    | Accessible, fun avec 1 ami          |
| 4v4    | Equipe      | HAUTE    | Format principal, tactique          |
| 8v8    | Grand Combat| MOYENNE  | Spectaculaire, moins competitive    |
| 1v1    | Solo        | BASSE    | Optionnel, desequilibre de classes  |

Recommendation: Lancer avec 2v2 et 4v4, ajouter 8v8 plus tard

SAISONS
-------
- Duree: 3 mois (4 saisons par an)
- Noms: Saison du Printemps, de l'Ete, de l'Automne, de l'Hiver
- Reset: Soft reset ELO entre saisons
- Rewards: Distribues a la fin de chaque saison

INSCRIPTION
-----------
Pre-requis pour ranked:
- Niveau 20 minimum (subclass debloquee)
- 10 matchs casual joues dans le format
- Compte verifie (email/telephone)

Formations d'equipe:
- 2v2: Duo fixe ou matchmaking solo-duo
- 4v4: Equipe pre-faite ou matchmaking
- 8v8: Toujours matchmaking (trop dur de trouver 8)


================================================================================
                           2. FORMAT 2V2 - DUO
================================================================================

CONCEPT
-------
Le format le plus accessible. Ideal pour jouer avec un ami.
Skill individuel + synergie de base.

CONFIGURATION ARENA
-------------------
Grille: 4x4 (chaque equipe a 2x4)
Spawn: Colonnes opposees
Temps max: 6 minutes

   EQUIPE A                    EQUIPE B
   +--+--+--+--+              +--+--+--+--+
   |A1|  |  |  |              |  |  |  |B1|
   +--+--+--+--+              +--+--+--+--+
   |A2|  |  |  |              |  |  |  |B2|
   +--+--+--+--+              +--+--+--+--+

REGLES
------
Consommables:
- 1 Potion de soin autorisee (cooldown 60s)
- 0 Elixir de buff (pas de prep)
- Nourriture: Desactivee

Resurrection:
- Pas de resurrection
- Une fois mort = out

Cooldowns:
- Reset au debut du match
- Aucun pre-buff autorise

Victoire:
- Eliminer les 2 adversaires
- OU avoir plus de % HP combine a la fin du timer
- Egalite: Overtime 1 minute, puis mort subite

COMPOSITIONS VIABLES 2V2
------------------------
Le meta 2v2 favorise les duos complementaires:

S-Tier (Meta):
1. Healer + DPS (Warrior ou Rogue)
   - Le classique. Healer keep alive, DPS kill.

2. Double DPS (Rogue + Mage)
   - Burst combo. Kill avant qu'ils reagissent.

A-Tier (Viable):
3. Tank + Healer
   - Unkillable. Gagne au timer.

4. Warrior + Archer
   - Front + Back synergy. Control + damage.

B-Tier (Situationnel):
5. Double Healer
   - Tres long, mais gagne souvent au timer.

6. Artisan + X
   - Dependra des skills artisan (support?)

STRATEGIES COMMUNES
-------------------
- Focus Healer: Tuer le healer en premier
- Peel: Proteger son partenaire du focus
- Trading: Echanger des CD avantageusement
- Positioning: Utiliser la grille 2D a son avantage

GAINS ELO 2V2
-------------
| Resultat     | ELO Gagne/Perdu |
|--------------|-----------------|
| Victoire     | +20 a +30       |
| Defaite      | -15 a -25       |
| Victoire vs +200 ELO | +35    |
| Defaite vs -200 ELO  | -30    |


================================================================================
                        3. FORMAT 4V4 - EQUIPE
================================================================================

CONCEPT
-------
Le format principal et le plus competitif.
Necessite composition, coordination, et strategie.

CONFIGURATION ARENA
-------------------
Grille: 4x8 (chaque equipe a 4x4)
Spawn: Zones de spawn protegees (3s invuln)
Temps max: 10 minutes

   EQUIPE A                              EQUIPE B
   +--+--+--+--+--+--+--+--+--+--+--+--+--+--+--+--+
   |A1|A2|  |  |  |  |  |  |  |  |  |  |  |  |B1|B2|
   +--+--+--+--+--+--+--+--+--+--+--+--+--+--+--+--+
   |A3|A4|  |  |  |  |  |  |  |  |  |  |  |  |B3|B4|
   +--+--+--+--+--+--+--+--+--+--+--+--+--+--+--+--+
   |  |  |  |  |  |  |  |  |  |  |  |  |  |  |  |  |
   +--+--+--+--+--+--+--+--+--+--+--+--+--+--+--+--+
   |  |  |  |  |  |  |  |  |  |  |  |  |  |  |  |  |
   +--+--+--+--+--+--+--+--+--+--+--+--+--+--+--+--+

REGLES
------
Consommables:
- 2 Potions de soin par joueur (cooldown 45s)
- 0 Elixir de buff
- Nourriture: Desactivee

Resurrection:
- 1 resurrection possible par equipe (total)
- Cast time: 6 secondes
- Ressuscite a 30% HP
- Cooldown: 90 secondes apres utilisation

Cooldowns:
- Reset au debut
- Pas de pre-buff

Victoire:
- Eliminer les 4 adversaires
- OU avoir plus de joueurs vivants a la fin du timer
- Egalite joueurs: % HP total
- Egalite parfaite: Overtime puis mort subite

COMPOSITIONS 4V4
----------------
La composition standard suit le schema:
1 Tank + 1 Healer + 2 DPS

Variantes viables:

A) Composition Standard
   - Tank (Warrior)
   - Healer
   - Mage (DPS)
   - Rogue ou Archer (DPS)

B) Double Healer ("Cleave")
   - Tank (Warrior)
   - 2 Healers
   - 1 DPS burst

C) Full Aggro ("Zerg")
   - 4 DPS
   - Pas de healer
   - Objectif: Kill fast ou lose fast

D) Turtle
   - 2 Tanks
   - 1 Healer
   - 1 DPS
   - Gagne au timer

ROLES DETAILLES
---------------

Tank (Warrior):
- Position: Front line
- Role: Absorber les degats, CC, peel pour healer
- Stats prioritaires: HP, DEF, CC resist
- Skills cles: Taunt, Shield, Charge

Healer:
- Position: Back line, protege
- Role: Maintenir l'equipe en vie
- Stats prioritaires: MAG, HP, Cast speed
- Skills cles: Group heal, Shield, Dispel

DPS Melee (Rogue):
- Position: Flanks, dive
- Role: Assassiner les cibles prioritaires
- Stats prioritaires: ATK, Crit, Speed
- Skills cles: Stealth, Burst combo, Escape

DPS Range (Mage/Archer):
- Position: Back line
- Role: Damage constant, CC
- Stats prioritaires: MAG/ATK, Crit, Range
- Skills cles: AOE, Snare, Execute

STRATEGIES 4V4
--------------

1. Target Calling
   - Un joueur (shotcaller) designe la cible
   - Toute l'equipe focus cette cible
   - Communication essentielle

2. Peeling
   - Proteger le healer des dives
   - CC les assaillants
   - Le tank doit peel, pas chase

3. Cooldown Trading
   - Forcer les CD defensifs adverses
   - Switcher de cible apres un gros CD
   - Punir les erreurs

4. Zone Control
   - Controler le centre de l'arene
   - Forcer les adversaires dans les coins
   - Utiliser les AOE strategiquement

GAINS ELO 4V4
-------------
| Resultat     | ELO Gagne/Perdu |
|--------------|-----------------|
| Victoire     | +25 a +35       |
| Defaite      | -20 a -30       |
| Victoire vs +200 ELO | +45    |
| Defaite vs -200 ELO  | -35    |

Note: Le 4v4 est le format avec les gains les plus eleves
car c'est le format principal.


================================================================================
                       4. FORMAT 8V8 - GRAND COMBAT
================================================================================

CONCEPT
-------
Le format spectaculaire. Moins competitif mais plus fun.
Chaos organise avec beaucoup d'action.

CONFIGURATION ARENA
-------------------
Grille: 4x10 par equipe (total 4x20)
Spawn: Zones protegees aux extremites
Temps max: 12 minutes

Zone speciale: Point de controle au centre
(Objectif secondaire pour encourager les affrontements)

REGLES
------
Consommables:
- 3 Potions par joueur
- Nourriture: Buff pre-match autorise

Resurrection:
- Illimitee (respawn apres 20 secondes)
- Respawn dans la zone de spawn
- Mort = -1 point pour l'equipe

Points:
- Kill: +1 point
- Controle du centre (30s): +1 point
- Timer: 12 minutes
- Victoire: Equipe avec le plus de points

COMPOSITIONS 8V8
----------------
Moins rigide que le 4v4. Suggestions:

Composition equilibree:
- 2 Tanks
- 2 Healers
- 4 DPS (mix melee/range)

Composition Deathball:
- 3 Tanks
- 3 Healers
- 2 DPS
- Reste groupe, avance ensemble

Composition Split:
- 2 groupes de 4
- Attaque sur 2 fronts
- Plus mobile

OBJECTIFS
---------
1. Eliminations
   - +1 point par kill
   - Focus les cibles isolees

2. Controle de Zone
   - Tenir le centre 30s = +1 point
   - Force les teamfights

3. Survie
   - Chaque mort = -1 point pour l'equipe
   - Pas de feed!

GAINS ELO 8V8
-------------
| Resultat     | ELO Gagne/Perdu |
|--------------|-----------------|
| Victoire     | +15 a +20       |
| Defaite      | -10 a -15       |

Note: Gains reduits car le format est plus chaotique
et moins skill-dependent.


================================================================================
                     5. SYSTEME DE RANKING UNIFIE
================================================================================

TIERS ET DIVISIONS
------------------
Du plus bas au plus haut:

| Tier      | ELO Range   | Icone  | Population |
|-----------|-------------|--------|------------|
| Bronze    | 0 - 999     | Bronze | ~25%       |
| Silver    | 1000 - 1499 | Argent | ~35%       |
| Gold      | 1500 - 1999 | Or     | ~25%       |
| Platinum  | 2000 - 2499 | Platine| ~10%       |
| Diamond   | 2500 - 2999 | Diamant| ~4%        |
| Master    | 3000+       | Maitre | ~1%        |

Chaque tier (sauf Master) a 4 divisions: IV, III, II, I
Exemple: Gold IV → Gold III → Gold II → Gold I → Platinum IV

PLACEMENT
---------
Debut de saison:
1. 10 matchs de placement obligatoires
2. Nouveau joueur: Commence a 1200 (Silver II)
3. Joueur existant: Soft reset = (ancien_elo + 1200) / 2
4. Performance des placements affecte le rang initial

Exemple de soft reset:
- Ancien: 2500 (Diamond IV)
- Nouveau: (2500 + 1200) / 2 = 1850 (Gold I)
- Doit remonter

DECAY (Inactivite)
------------------
| Tier      | Inactivite avant decay | Perte/semaine |
|-----------|------------------------|---------------|
| Bronze    | Jamais                 | 0             |
| Silver    | 2 semaines             | -10           |
| Gold      | 1 semaine              | -15           |
| Platinum  | 1 semaine              | -20           |
| Diamond   | 4 jours                | -25           |
| Master    | 3 jours                | -30           |

Prevention: Jouer 1 match ranked suffit a reset le timer

PROMOS ET DEMOS
---------------
Promotion (monter de division):
- Atteindre le seuil ELO
- Pas de "promo series" (simplification)
- Promotion immediate

Demotion (descendre):
- Tomber sous le seuil ELO
- Protection: 3 matchs de grace apres promo
- Apres protection: demotion immediate

Exemple:
- Gold III = 1625 ELO
- Gold II = 1750 ELO
- Atteindre 1750 = Promotion Gold II
- Tomber a 1624 (apres protection) = Demotion Gold III


================================================================================
                          6. MATCHMAKING
================================================================================

ALGORITHME
----------
Objectif: Matchs equilibres avec temps d'attente raisonnable

Parametres:
1. ELO (principal)
2. Winrate recent (anti-smurf)
3. Temps d'attente (flexibility)

Expansion de range:
| Temps attente | Range ELO |
|---------------|-----------|
| 0-30s         | +/- 100   |
| 30s-1min      | +/- 150   |
| 1-2min        | +/- 200   |
| 2-3min        | +/- 300   |
| 3-5min        | +/- 400   |
| 5min+         | +/- 500   |

GROUPES PRE-FAITS
-----------------
2v2:
- Duo autorise
- Solo queue dans le meme pool

4v4:
- Groupes de 2, 3, ou 4 autorises
- Solo queue: forme avec autres solos
- Groupes pre-faits vs groupes pre-faits si possible

8v8:
- Pas de groupe pre-fait (trop difficile)
- Toujours matchmaking solo

RESTRICTION DE GROUPE
---------------------
- Ecart ELO max entre membres: 500
- Exemple: Un Diamond (2500) ne peut pas queue avec un Bronze (500)

ANTI-SMURF
----------
Detection:
- Winrate > 70% sur 20 games = suspect
- KDA anormalement eleve = suspect
- Score de skill interne calcule

Mesures:
- Gains ELO acceleres pour smurfs detectes (+50%)
- Placement initial ajuste si performance exceptionnelle
- Matchmaking contre joueurs similaires

================================================================================
                       8. ANTI-CHEAT & FAIRPLAY
================================================================================

ANTI-CHEAT
----------
Detection automatique:
- Input impossible (actions trop rapides)
- Position anormale (teleport)
- Degats impossibles (overflow)
- Patterns de bot

Sanctions:
- 1ere offense: Warning + review
- 2eme offense: Ban 7 jours + reset ELO
- 3eme offense: Ban permanent

FAIRPLAY
--------
Comportement toxique:
- Chat abuse: Mute 24h → 7j → permanent
- AFK intentionnel: -50 ELO + timeout 30min
- Grief (feed intentionnel): Review manuelle

Report system:
- Bouton report en fin de match
- Categories: Cheat, AFK, Toxic, Grief
- Review automatique + manuelle si nombreux reports

INTEGRITY
---------
Win trading detection:
- Patterns de matchs suspects
- Memes joueurs qui s'affrontent trop souvent
- Resultats previsibles

Sanctions win trading:
- Reset ELO des deux parties
- Ban temporaire ou permanent

                      9. META ET COMPOSITIONS
================================================================================

META ANTICIPEE PAR FORMAT
-------------------------

2V2 META:
- Tier S: Healer + Rogue (burst + sustain)
- Tier S: Healer + Warrior (control + sustain)
- Tier A: Double DPS (Mage + Rogue)
- Tier B: Tank + Healer (long mais safe)

4V4 META:
- Tier S: Warrior + Healer + Mage + Rogue
- Tier S: Warrior + 2 Healers + DPS burst
- Tier A: 2 Warriors + Healer + Mage
- Tier B: Full DPS (risky mais rewarding)

8V8 META:
- Deathball (groupe compact) vs Split (2 groupes)
- Depend de la coordination de l'equipe
- Moins de meta fixe, plus de chaos

COUNTERS
--------
| Composition      | Counter             | Pourquoi                    |
|------------------|---------------------|-----------------------------|
| Healer + DPS     | Double DPS burst    | Kill avant heal             |
| Double Healer    | Mortal Strike comp  | Heal reduction              |
| Tank Heavy       | Sustained DPS       | Outlast the mana            |
| Full DPS         | Tank + Healer       | Survive the burst           |

SKILLS PVP IMPORTANTS
---------------------
A prioriser pour le PvP:

CC:
- Stun (hard CC)
- Silence (anti-caster)
- Root (anti-melee)
- Slow (kiting)

Defensifs:
- Shield
- Heal
- Invuln (cooldown long)
- CC break

Offensifs:
- Burst combo
- Execute (bonus dmg low HP)
- AOE (cleave)
- Purge (remove buffs)
